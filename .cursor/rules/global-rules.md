# Глобальные правила для AI помощника

## Поведение
- Всегда показывать код который хочешь изменить
- Спрашивать разрешение перед изменениями
- Объяснять что делает код (для новичков)
- Комментарии и объяснения на русском языке
- Код и переменные на английском

## Стиль кодирования
- Подробные комментарии
- Читаемый код
- Следовать принципам SOLID
- Использовать современные практики
